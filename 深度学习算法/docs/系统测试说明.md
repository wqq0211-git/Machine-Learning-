# 系统测试说明

## 测试环境

Windows、Python 3.10/3.11、Node.js 18+。CUDA 可选。

## 后端测试

```powershell
cd backend
python -m compileall .
pytest
```

覆盖 `/api/health`、`/api/models`、`/api/dataset`、非法模型名和非图片上传。

## 前端测试

```powershell
cd frontend
npm install
npm run build
```

检查路由、组件导入、ECharts 初始化和生产构建。

## 异常测试

- 权重不存在：服务可启动，预测返回 503。
- 非图片上传：返回 400。
- 批量超过 20 张：返回 400。
- 无评估结果：前端显示暂无评估结果。

## 已知限制

正式准确率、AUC 和混淆矩阵必须在训练并评估后生成；初始 JSON 只表示未训练状态。
