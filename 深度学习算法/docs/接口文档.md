# 接口文档

统一前缀：`/api`。统一响应：

```json
{"code": 200, "message": "success", "data": {}}
```

## GET /health

返回服务状态、Python、PyTorch、设备、CUDA 和模型加载状态。

## GET /models

返回 CNN 与 ResNet-18 的模型说明、输入尺寸、权重是否存在、是否可用、参数量。

## POST /predict

`multipart/form-data`：`file`、`model_name`。返回类别、置信度、Top-3、十类概率和推理耗时。

## POST /predict/compare

上传一张图片，同时返回 CNN 和 ResNet-18 预测结果、一致性、置信度差值。

## POST /predict/batch

字段：`files`、`model_name`。最多 20 张，返回成功数、失败数和每张图片结果。

## GET /metrics

读取 `backend/results/metrics.json`。

## GET /training-history

读取 `backend/results/training_history.json`。

## GET /confusion-matrix/{model_name}

读取指定模型混淆矩阵。

## GET /class-metrics/{model_name}

读取指定模型每类 precision、recall、F1。

## GET /dataset

返回 CIFAR-10 数据集基本信息和类别中英文名称。
