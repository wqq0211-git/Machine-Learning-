# 系统设计说明

## 需求分析

系统用于展示 CIFAR-10 图像分类任务中自定义 CNN 与 ResNet-18 的训练、评估和推理对比。

## 总体架构

前端 Vue 负责页面交互和图表展示；后端 FastAPI 提供 `/api` 接口；PyTorch 负责模型定义、训练、评估和推理。

## 前端架构

前端包含首页、单图识别、批量预测、模型对比、数据集分析和项目说明。Axios 统一封装请求，ECharts 负责指标、曲线和混淆矩阵展示。

## 后端架构

后端按 router、model_manager、preprocessing、prediction 拆分。模型启动时加载一次，权重缺失不会阻止服务启动。

## 训练流程

训练脚本下载 CIFAR-10，按 90%/10% 划分训练集和验证集，保存最佳权重、最后权重、训练历史和模型信息。

## 部署流程

先安装 Python 和 Node 依赖，再分别启动 FastAPI 与 Vite。Windows 可使用 `start_all.bat` 同时打开前后端。
